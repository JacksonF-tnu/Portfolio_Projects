# Jackson Fuller Portfolio Website

This is a simple portfolio website built from the information in `Portfolio_Presentation.pptx`.

## Files
- `index.html` — main website content
- `styles.css` — design and responsive layout
- `script.js` — mobile menu and dynamic year

## How to use
1. Open `index.html` in a browser to preview the site.
2. Replace the placeholder `#` links with your real LinkedIn, GitHub, and project repository URLs.
3. Upload the folder to GitHub Pages, Netlify, Vercel, or another hosting service.

## Suggested edits
- Add screenshots of your projects.
- Add a downloadable resume PDF link.
- Replace placeholder repository links with actual GitHub URLs.
